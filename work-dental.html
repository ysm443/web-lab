コンセプト設計にあたり考えたこと

医療サイトに求められる「清潔感」と「信頼性」を軸に、初診患者が感じる不安を取り除く設計を考えました。過剰な演出を排し、必要な情報を迷わず見つけられることを優先。24時間 WEB 予約という行動を自然な流れで促す導線設計に注力しています。

課題
初診患者の来院ハードルを下げ、清潔感・信頼感のある医院イメージを確立しながら 24時間 WEB 予約への誘導を強化する
コンセプト
清潔感・アクセス・24時間予約を最初に伝え、安心感を積み重ねながら自然に WEB 予約へ誘導
ターゲット
20〜60代・港区芝田町駅周辺のビジネスパーソン・ファミリー層
情報設計
清潔感・アクセス・24時間予約を最初に伝え、安心感を積み重ねながら WEB 予約へ誘導
デザイン
信頼の青×ホワイト。控えめな演出で安心感を維持。過剰アニメーションは避ける
トーン＆マナー
クリーン・ミニマル。余白多め・情報密度低め
